Что выведет программа? Объяснить вывод программы.

```go
package main

func main() {
	ch := make(chan int)
	go func() {
		for i := 0; i < 10; i++ {
			ch <- i
		}
	}()

	for n := range ch {
		println(n)
	}
}
```

Ответ:
```
Будут выведены значения от 0 до 9 включительно и затем произойдет deadlock.
Так как после отправки всех значений канал, итерируемый с помощью range,
не закрывается, то такое использование приведет к вечной блокировке.

Для корректной работы требуется закрыть канал после блока for в анонимной
горутине.
```
